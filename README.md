# python协同过滤算法在线课程推荐系统

#### 介绍
python基于协同过滤算法教学资源推荐系统，采用基于流行度的热点推荐，推荐点击量较多的课程；同时采用基于用户与基于物品的协同过滤推荐算法，如果基于用户与基于物品的协同过滤推荐算法均没有推荐结果，采用兴趣标签推荐，随机查询当前登录用户的兴趣标签中的课程，同时过滤当前登录用户已经评分、收藏、点赞的课程。数据分析，数据爬虫
